# dv2_insurance

Data visualization project focused on insurance data.

## Project Goal

The project explores an insurance dataset and presents the results through notebook-based visualizations and dashboard-style outputs. The focus is on making the main patterns in the data easier to inspect and communicate.

## Files

- `artyom_ashigov_insurance.ipynb` - full notebook with data loading, exploration, visualizations, and dashboard work.
- `insurance_data.csv` - source dataset used in the analysis.

## Analysis Topics

- Insurance dataset structure and variable review.
- Exploratory data visualization.
- Visual summaries for comparing groups and patterns.
- Dashboard-ready charts and presentation outputs.

## Tools

Python, pandas, Jupyter Notebook, and visualization libraries.
